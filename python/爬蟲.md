# 使用urllib
```python
import urllib.request as req

url = 'https://www.ptt.cc/bbs/LoL/index.html'

res = req.urlopen(url)

result = res.read().decode('utf-8')
```

# 帶入header
通常在請求取得網頁資料的時候會需要同時帶入一些額外資訊，才能通過該網站的驗證且順利取得資料。

通常會帶入的資料有以下

- Cookie
- User-Agent
- Accept
- Accept-Language
